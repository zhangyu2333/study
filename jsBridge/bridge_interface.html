通用接口列表

一、namespace=app

1. 唤醒APP登录

chelunJSBridge://app/login
传递的参数：
loginCallBackName 点击登录按钮之后的回调
loginCallBackName中返回的参数：
result 1 登录成功/其他 取消登录
message 描述
data {token: '' ,uid:'',nickname:'',avatar:'头像地址'}

2. 查询单个应用是否已安装（包名查询）

chelunJSBridge://app/isAppInstalled
传递的参数：
os ios/android
name IOS传scheme android传包名
callback中返回的参数：
result 1 成功
message 描述
data {code:1安装 -1未安装 }

3. 使用scheme(ios)或者包名(andorid)启动一个app

chelunJSBridge://app/launchApp
传递的参数：
name IOS传scheme android传包名
callback中返回的参数：
data {code:1安装 -1未安装 }

4. 打开schema（客户端支持的链接格式）

chelunJSBridge://app/openSchema
传递的参数：
schema 带协议的链接
callback中返回的参数：
data {code:1打开成功 -1打开失败 }

5. 关闭当前web页面

chelunJSBridge://app/closeWebView

6. 网络请求接口（GET）

chelunJSBridge://app/fetch
传递的参数：
params json格式
headers json格式
fetchCallbackName 回调函数
fetchCallbackName中返回的参数：
data {body: response body}

7. 获取网络状态

chelunJSBridge://app/netStatus
callback中返回的参数：
data {code: 1 移动网络 2 WIFI -1没有获取到}

8. 唤起app支付

chelunJSBridge://app/pay
传递的参数：
price 价格
orderNum 订单号
channels 支付渠道 weixin,alipay
callbackUrl 回调网址，需要提前encode 支付后会跳转带上支付状态，参数名 status {0失败，1成功} (第三方使用)
payCallback 支付结束后回调 {result: 1，或其他值表示失败 } （前端SDK使用）

二、namespace=ui

1. 使用native分享接口, 需要将分享的内容提前设置到window全局变量中

chelunJSBridge://ui/shareMessage?to=clMessage

将to的参数替换就会变成其它分享
clMessage：车轮车友，wxTimeline：微信朋友圈
wxMessage：微信好友，qq:QQ，sina：新浪微博，sms：短信
传递的参数：
to 分享渠道
shareCallBackName 分享之后的回调函数
shareCallBackName中返回的参数：
result 1 成功
message 描述

2. 使用浏览器打开

chelunJSBridge://ui/openWithBrowser

传递的参数：
url 网址，需要提前encode

3. 使用native刷新

chelunJSBridge://ui/refresh

4. 使用native绑定微信

chelunJSBridge://ui/bindWeixin

传递的参数：
bindCallBackName 回调函数

5. 使用native绑定手机号

chelunJSBridge://ui/bindPhone

传递的参数：
bindCallBackName 回调函数
bindCallBackName返回的参数：
data {phone:手机号}
如果用户已经绑定了手机号，那客户端直接调用callback函数

6. 二维码扫描

chelunJSBridge://ui/scanQRCode

传递的参数：
scanCallBackName 回调函数
scanCallBackName返回的参数：
data {value:扫描出来的内容}

7. 复制文字到剪切版

chelunJSBridge://ui/copy
传递的参数：
text 需要复制的内容

8. 显示隐藏标题栏

chelunJSBridge://ui/showTitleBar
chelunJSBridge://ui/hideTitleBar

9. 触发震动

chelunJSBridge://ui/vibration
传递的参数：
milliseconds 时长
vibrationCallbackName 回调函数
vibrationCallbackName返回的参数：
result 1：结束

10. 摇一摇

chelunJSBridge://ui/shakeItOff

传递的参数：
shakeCallbackName 回调函数
shakeCallbackName返回的参数：
result 1：结束

11. 日期选择控件

chelunJSBridge://ui/datepicker
传递的参数：
minDate 最小日期

由yyyy-mm-dd表示 系统默认
maxDate 最大日期

由yyyy-mm-dd表示 系统默认
curDate 选中日期

由yyyy-mm-dd表示 默认当前日期
pickCallbackName 回调函数
pickCallbackName返回的参数：
data {date:'2015-05-05'}

如格式 yyyy-mm-dd 匹配 2016-05-03

12. 时间选择控件

chelunJSBridge://ui/timepicker
传递的参数：
minTime 最小时间

由h:i表示 系统默认
maxTime 最大时间

最大日期，由h:i表示 系统默认
curTime 选中时间

由h:i表示 默认当前时间
pickCallbackName 回调函数
pickCallbackName返回的参数：
data {date:'12:33'}

如格式h:i 匹配11:40

13. 下拉刷新控件

注：客户端只提供UI和交互效果，刷新逻辑由H5自行控制

chelunJSBridge://ui/pullRefresh
传递的参数：
isRefresh

true：开启刷新

false：关闭刷新

当刷新完成后可发起关闭下拉刷新效果的请求
onRefreshCallback 回调函数
onRefreshCallback返回的参数：
result 1：结束

14. 使用内部浏览器打开

chelunJSBridge://ui/openInnerBrowser
传递的参数：
url 网址，需要提前encode

15. 修改标题

chelunJSBridge://ui/setTitle
传递的参数：
title 标题

16. 添加标题栏右上角按钮 （替换原有的按钮）

chelunJSBridge://ui/setTitleBarMenu
传递的参数：
btnTitle 按钮文案
btnImg 按钮图片 外网可以访问的图片url，100*100即可 图片与文案取其一
callbackFun 点击按钮响应的函数 （开放给第三方使用，前端SDK忽略此参数）
clickCallback 点击按钮响应（前端SDK使用）

17. 扫描车牌

chelunJSBridge://ui/scanCarNo
传递的参数：
needUpload 是否需要上传图片
scanCallBackName ｛result: 1扫描成功, data : { carNo:'陕a440u9', transId: "DJUe343432DF"}}

三、namespace=data

1. 查询是否绑定手机号

chelunJSBridge://data/hadBoundPhone

返回的参数：
data {code:1已绑定 -1未绑定}

2. 强制客户端到服务端验证token（loading由前端处理），成功或失败都会回调

chelunJSBridge://data/verifyToken

传递的参数：
verifyCallbackName 绑定回调
verifyCallbackName返回的参数：
data {token:有效则返回，前端更新token为此token}

四、namespace=device

1. 选取图片 ，目前只支持单张图片

chelunJSBridge://device/chooseImage

传递的参数：
type

0 同时拍照和相册选择【默认】

1 拍照 2相册选择
chooseCallbackName 绑定回调
chooseCallbackName返回的参数：
data {url:图片url}

2. 获取定位

chelunJSBridge://device/getLocation

传递的参数：
locationCallBackName 绑定回调
locationCallBackName返回参数data的内容：
gcjLat 纬度
gcjLng 经度
gaodCityCode 高德城市编号
gdAdCode 高德省份编号
country 国家
province 省份
city 城市
district 区、县
address 地址
fullAddressOnlyCity 从城市开始的完整地址
